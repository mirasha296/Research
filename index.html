Exploring Digital Color Theory:

CMYK vs. RGB in an Interactive Virtual Environment

This project explores how color changes based on perspective within an immersive virtual reality environment. The focus is a three-dimensional CMYK cube that looks completely different from every angle. In reality, as the viewer moves it around, rotates it, or shifts position, new color combinations appear and others disappear. The goal is to recreate this experience in VR so that users can physically navigate the cube and observe how subtractive color relationships shift dynamically in space.

Traditional color theory is often taught using flat diagrams or static models. While helpful, they do not fully capture how color can behave in a spatial, interactive environment depending on angle and light fixtures. By building a virtual version of the CMYK cube, this project moves color theory into a 3D virtual environment that introduces new challenges, particularly in visualizing the differences between subtractive and additive color systems. Each angle reveals a new visual outcome, reinforcing that color is not only mathematical but also spatial and experiential.

Through interactive visualization and immersive design, this research connects digital media, perception, and technology. The project demonstrates how virtual reality can deepen understanding of complex visual systems by allowing individuals to explore color theory from another lens.
